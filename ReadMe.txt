#-----------适用的单片机介绍------------#
	项目基于宏晶科技公司的STC15W4K48S4，代码适用于所有STC15W4K系列单片机
	代码编写于Keil C51 uVision5 Version 9.54
	基本性能：
	4096字节片内RAM数据存储器
	宽电压：2.5V～5.5V
	ISP编程时内部时钟从5MHz~30MHz可设
	16/32/40/48/56/58K/61K/63.5K字节片内Flash程序存储器
	高速ADC，8通道10位
	6通道15位PWM
	7个定时器/计数器
	2路CCP
	四串口/UART
	SPI高速同步串行通信接口
	硬件看门狗

#----------Questions&Answers------------#

Q：为什么会有这个坑？
A：学校组织了一个循迹智能车比赛，出于锻炼目的而参加。

Q：比赛时间？
A：2015年10月中旬开始预热，比赛时间为2015年12月11日，实际准备的时间大概有一个月。

Q：比赛结果？
A：由于本人太小看了这个项目开发的难度，挖的坑太大了，没有完赛。其它队伍使用主办方已经写好
的程序和设计好的车子/电路，只需进行焊接组装，加以调试就可以进行比赛了，我们基本得从头做起。

Q：感想？
A：虽然没能参加比赛，但是为了这个比赛写出的代码量比几年学习下来写的总代码量还要多，对于以
学习锻炼目而参加比赛的我来说，只赚不亏，至于这个坑未完工，以后还有机会填的。

Q：介绍小车？
A：我们使用的小车是用一台遥控玩具越野车进行改装，车的长宽高大概为250*180*60，车子为单电机
四驱车，前轮使用一个16g舵机进行转向控制。前后轮都有弹簧避震系统，并且配有机械差速箱。车子
的驱动电机额定电压为7.4V,额定空转转速大概为1.5WRPM，车子在遥控下最高时速为45Km/h，即
12.5m/s，性能强悍（然并卵）。

Q：改装方向？
A：
1.拆除遥控装置，改换由单片机组成的自动控制电路。
2.原配的舵机为五线舵机，不符合我们的改装要求，故换成同规格的三线舵机。
3.电机驱动使用L298N驱动模块，通过单片机的PWM进行转速控制。速度检测有两套方案，一是使用增量
式旋转编码器，二是使用霍尔传感器，钢磁固定在轮子上。
3.循迹检测使用红外传感器，通过检测一排传感器的电阻变化不同坦直黑线位置。
4.电源方面使用1100mah的2S电池，除了直接给电机供电外，还配合LM2596降压稳压模块给车的其它部件
进行供电。
5.小车还配有HC-05蓝牙模块方便数据回传和脱机调试。

Q：项目进程？
A：先是进行单片机模块程序的编写。使用的STC15W4K48S4单片机并不像Arduino那样有那么多现成的库，
一切都要从数据手册从头写起。首先完成的是用于通讯串口库IO.h(我知道应该叫serial.h的),然后是
AD模块的库ADC.h，用于红外接收模块的电压检测(这个日后细说)，第三个是用于电机驱动模块的PWM
模块PWM.h，第四个模块和第五个模块分别是舵机的控制模块STEERING.h和电机的控制模块engine.h，
然后是用于捕获编码器和霍尔脉冲的PCA模块PCA.h。最后到了整合各基础模块的INFRARED.h,RoadWatch.h
和SMARTCAR.h。写到最后几个模块已经临近比赛了，实在没有写完，想实现的各种功能也都变成了深坑。
